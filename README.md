### Game
2D fighting game built as a pre-university project. Three levels where you fight against three unique types of enemies which attack in different ways. Features: state-driven systems, animation from sprite sheets, collision detection, leaderboard, audio management, and a transformation the player can access to turn the tide. 

Audio originally sourced from itch.io, then edited/mixed in Audacity. Character sprites sourced from itch.io. Images used to construct level scenery sourced from itch.io.
